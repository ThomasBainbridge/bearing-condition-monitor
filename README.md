# Bearing Condition Monitor

A machine learning and signal-processing project for bearing fault diagnosis using vibration data.

## Goal
Build an engineering-focused condition monitoring tool that uses vibration signals to identify bearing health state and fault type.

## Planned Version 1
- Load and inspect raw bearing vibration data
- Plot time-domain and frequency-domain signals
- Extract diagnostic signal features
- Train baseline ML models for fault classification
- Build a simple interface for predictions
